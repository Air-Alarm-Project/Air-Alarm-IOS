# Air-Alarm iOS

## 개발환경

- Mac OS Big Sur 11.6
- Adobe XD
- Xcode 13
- Swift Ui
- iOS 15
- iPhone 12 mini, iPhone 12, iPhone 12 Pro, iPhone 12 Pro MAX, iPhone SE2

## 파일/폴더 안내
Air-Alarm-IOS/Air-Alarm
- /API : Rest API 소스
- /ChartView : 그래프 정보
- /Login : 로그인
- /Signup : 회원가입
- /Notification : 알림구현
- /Setting : 설정창
- /Tab : 메인 탭뷰
- /View : 메인화면 뷰 (실시간/그래프/지역)
